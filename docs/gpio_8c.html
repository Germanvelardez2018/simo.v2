void simo_gpio_set_callback_irq ( uint8_t  gpio_pin,
uint32_t  event,
gpio_irq_callback_t  funct_callback,
bool  enable 
)

Configuro una funcion de interrupcion.

Parámetros
gpio_pingpio_pin seleccionado
eventEs el uint32_t que se le pasa a la function callback cuando surge la irq.
funct_callbackfuncion que maneja la interrupcion
enableSe habilitan las interrupcion. Si es falso la funcion configura pero no habilita la interrupion
Nota
funct_callback : void (*gpio_irq_callback_t)(uint gpio, uint32_t events);
El valor event que se envia a la function depende del evento asociado: 
   bit | interrupt
   ----|----------
     0 | Low level
     1 | High level
     2 | Edge low
     3 | Edge high

** void
